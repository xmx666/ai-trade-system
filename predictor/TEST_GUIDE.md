## 📊 标准预测参数

根据krnos模型的标准预测方式：

- **历史数据步数**: 450步
- **预测步数**: 120步
- **蒙特卡罗模拟**: 10次（限制）

## ⚙️ 测试配置

### 数据要求

- **历史数据**: 至少450步（标准）
- **最小数据**: 100步（最低要求）
- **数据格式**: 价格和成交量数组

### 预测参数

```python
HISTORY_STEPS = 450      # 历史数据步数
PREDICTION_STEPS = 120   # 预测步数
N_SIMULATIONS = 10       # 蒙特卡罗模拟次数
```

## 🔍 验证要点

### 1. 预测结果格式

预测结果应包含：
- `trend`: 趋势方向（'up', 'down', 'sideways'）
- `trend_strength`: 趋势强度（0-1）
- `mean_prediction`: 预测曲线（120步）
- `confidence_interval_lower`: 置信区间下限
- `confidence_interval_upper`: 置信区间上限
- `price_range`: 价格范围
- `prediction_horizon`: 预测步数（120）
- `n_simulations`: 模拟次数（10）

### 2. 数据验证

- 历史数据长度 ≥ 450步
- 预测曲线长度 = 120步
- 价格和成交量数据长度一致
- 数据值在合理范围内

### 3. 性能验证

- 预测时间 < 30秒（使用10次模拟）
- 内存使用合理
- 缓存文件正确保存
