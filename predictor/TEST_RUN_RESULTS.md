## 🔍 故障排除

### 问题1: 代理连接失败

**检查**:
1. Clash是否启用"允许局域网连接"
2. 代理地址是否正确（`172.27.128.1:7890`）
3. Windows防火墙是否允许Clash接受连接

**解决**:
```bash
# 检查代理是否可访问
curl -x http://172.27.128.1:7890 https://fapi.binance.com/fapi/v1/time
```
