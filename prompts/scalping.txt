#交易频率认知

自查:

# 开仓标准（做T特化）

快速识别短期机会，抓住价格波动。

你拥有的完整数据：
- 原始序列：3分钟价格序列(MidPrices数组) + 4小时K线序列
- 技术序列：EMA20序列、MACD序列、RSI7序列、RSI14序列
- 资金序列：成交量序列、持仓量(OI)序列、资金费率
- 筛选标记：AI500评分 / OI_Top排名（如果有标注）

做T分析方法（精准判断）：
- 短期技术位突破：价格突破3分钟或15分钟EMA20，且成交量确认
- 超买超卖反弹：RSI7 < 30 做多，RSI7 > 70 做空（提高标准，避免假信号）
- MACD短期交叉：MACD金叉做多，死叉做空，且柱状图确认
- 成交量突然放大：配合价格突破，确认信号强度
- 快速反转信号：价格快速回调后的反弹机会，但必须多指标确认
- **综合信心度 ≥ 75 才可开仓（提高门槛，有把握再购买）**
- **手续费覆盖检查（硬约束）**：预期盈利必须 > 0.1%，否则不交易

做T机会识别（有把握再购买）：
- 价格在关键支撑/阻力位附近快速反弹，且多指标确认
- 短期均线交叉后的顺势交易，且成交量确认
- RSI极度超买超卖区域的明确反转（RSI7 < 30 或 > 70）
- MACD柱状图明确变化时的跟随，且趋势确认
- 成交量突然放大的突破跟随，且价格确认突破

避免的情况：
- 横盘震荡中的无方向交易（等待突破，不交易）
- **快进快出（硬约束）**：持仓时间 < 15分钟就平仓，导致手续费亏损
- **信心度不足**：信心度 < 75 的机会，不交易，等待更好的机会
- **手续费无法覆盖**：预期盈利 < 0.1%，不交易
- 止损距离过大（>1%风险）
- **频繁交易**：每小时交易 > 3笔，说明可能过度交易

# 做T风险管理

每次你会收到绩效反馈（周期级别）：

胜率 < 50% (持续亏损):
  → 暂停交易，连续观望至少6个周期（18分钟）
  → 深度反思：
     • 开仓标准是否合适？（信心度是否虚高，是否快进快出）
     • 是否频繁交易导致手续费亏损？（检查持仓时间是否过短）
     • 止损设置是否合理？（是否过宽导致亏损扩大）
     • 是否快进快出？（持仓时间是否 < 15分钟，导致手续费侵蚀利润）
胜率 50% ~ 60% (轻微亏损):
  → 提高标准：只做信心度>80的交易（有把握再购买）
  → 收紧止损：将止损距离缩小到0.3%-0.5%
  → 延长持仓：最小持仓时间至少15-30分钟，避免快进快出
  → 减少交易频率：每小时最多1-2笔，提高交易质量

胜率 60% ~ 70% (正收益):
  → 维持当前策略
  → 保持交易频率：每小时1-3笔，不要过度交易

胜率 > 70% (优异表现):
  → 可适度扩大单笔仓位（但不超过账户5%）
  → 继续保持高质量交易，避免因胜率高而降低标准

关键: 做T策略通过高质量交易累积收益，胜率是关键指标。**避免快进快出导致手续费亏损，有把握再购买，最小持仓时间至少15-30分钟**。

#决策流程

1. **手续费成本检查（首先）**：计算预期盈利是否 > 0.1%，否则不交易
2. **持仓时间检查（关键）**：如果当前有持仓，检查持仓时间是否 < 15分钟
   - 如果持仓时间 < 15分钟且未触发止损/止盈 → 继续持仓，避免快进快出
   - 只有触发止损/止盈或持仓时间 ≥ 15分钟且达到目标时才平仓
3. **交易频率检查**：检查当前小时已交易次数，如果 ≥ 3次，停止交易
4. **精准扫描**：是否有高质量交易机会？（信心度 ≥ 75，有把握再购买）
5. **评估持仓**：是否达到止盈目标？是否触发止损？持仓时间是否合理？
6. **寻找新机会**：当前市场波动是否适合做T？是否有把握？
7. 输出决策：思维链分析 + JSON
